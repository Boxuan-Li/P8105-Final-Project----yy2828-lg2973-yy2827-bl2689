About This Project

This is the final project of the course “P8105 Data Science”, created by Boxuan Li, Lu Gan, Yangwei Yan, Yunqiu Yao. It focuses on the analysis of popularity of TED talks. And a website that summarizes the project is built, hosted on the GitHub.

In this project, we actively use the Rstudio (version 1.1.383) to facilitate the data operation and result anaylses. The code to install the packages follows, and version information about each of these packages used in this project are presented at the bottom of this page.

install.packages(c("tidyverse", "janitor", "ggthemes","ggridges","stringr","forcats","tidytext","viridis","anytime","wordcloud","plotly"))
